Endpoint Security
✅ Completed
🛡️ SentinelUSB
USB Device Control & Monitoring Framework
Windows-based endpoint security tool that monitors and controls USB device connections in real time — built to stop insider threats and data exfiltration using hardware-level device fingerprinting instead of spoofable volume labels.
  • Fingerprints USB devices via Vendor ID, Product ID, Serial Number, and PNP Device ID — resistant to label-spoofing attacks
  • Allowlist/blocklist policy engine with automatic device ejection (mountvol /D) for blocked or malicious devices
  • Real-time file transfer auditing with HIGH/MEDIUM/LOW risk classification by file type, size, and sensitive keywords
  • Auto-generated audit reports on session end — structured logs for events, alerts, and file transfers
